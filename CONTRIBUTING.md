# Contributing

Thank you for your interest in contributing to Genetics Practice Engine 2.0.

## Mission

Our goal is to build a scientifically accurate, high-quality, open educational
genetics platform for undergraduate students.

## Core Principles

Every contribution should aim to be:

- Biologically accurate
- Educationally valuable
- Self-contained
- Clearly worded
- Appropriate for undergraduate genetics
- Easy to maintain

## Before Submitting

Please verify that:

- Questions contain no hidden assumptions.
- Dominance, inheritance pattern, linkage, penetrance, or epistasis are stated when required.
- Only one answer is clearly correct.
- Distractors are plausible.
- Explanations teach the reasoning rather than only revealing the answer.

## Coding Guidelines

- Keep modules focused on a single topic.
- Reuse shared utilities where possible.
- Avoid duplicating code.
- Keep functions small and well named.

## Topic Structure

Each topic should contain:

1. Learning objectives
2. Reasoning families
3. Parameterized generators
4. Feedback and explanations

## Reporting Issues

Bug reports should include:

- Topic
- Difficulty
- Steps to reproduce
- Expected behavior
- Actual behavior
- Screenshot (if applicable)

Thank you for helping improve genetics education!
